Go to definition
=================

How to
============
In IntraMine's Viewer, you can see a list of links to definitions for functions, and sometimes for classes and structs. To see the list in a Viewer-displayed source file, click on a function name and pause with your cursor remaining over the name: in a second or so you will then see one or more clickable links pop up.

How it works
=====================

You might sometimes be disappointed
========================
Support for C++ in particular is not very good, since the source isn't being parsed. For example if you have twenty different implementations of a method called readAll(), you'll see a list of the first five or so that were found, without regard for any relevance to your specific circumstances.
